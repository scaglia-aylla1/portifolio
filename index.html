Desenvolvimento de SPAs com React

Crio aplicações do tipo SPA (Single Page Application) com React.js, garantindo interfaces dinâmicas, rápidas e interativas. Ideal para dashboards, sistemas de controle, catálogos, entre outros projetos que exigem uma experiência de usuário fluida.

O que posso entregar:

  • Componentização eficiente com React

  • Gerenciamento de estado com useState e useEffect

  • Integração com APIs externas (RESTful)

  • Código organizado e otimizado para SEO

  • Estilização com CSS ou bibliotecas como Bootstrap/Tailwind

  • Deploy com Vercel, Netlify ou GitHub Pages

Api de Integração

Sistema de Integração de APIs com Spring Boot

Este projeto foi desenvolvido com foco em integrar múltiplas APIs externas de forma organizada e escalável, utilizando Spring Boot, Java e arquitetura em camadas. Ele simula um sistema robusto de integração que pode ser adaptado para diferentes cenários de consumo de dados externos.

A aplicação é estruturada com boas práticas de desenvolvimento, como o uso de DTOs, Services e Controllers bem definidos, além da injeção de dependência com Spring. O projeto demonstra minha capacidade de lidar com integrações REST e manipulação de dados em aplicações backend Java.

Back-End

Controle Financeiro

Sistema de Controle Financeiro Pessoal

Aplicação completa para gerenciamento de finanças pessoais, desenvolvida com Java, Spring Boot, JPA, MySQL e validações com Bean Validation. O sistema permite o cadastro e controle de receitas e despesas, com separação clara entre camadas e uso eficiente de boas práticas de POO.

O projeto demonstra habilidades tanto no backend quanto na organização de regras de negócio, utilizando anotações do Spring, Lombok para otimizar o código, e integração com banco de dados. É ideal para demonstrar domínio em CRUDs completos, boas práticas REST e persistência com JPA.

FullStack

Blog Pessoal

Blog Pessoal – Aplicação Fullstack com Java e React

Aplicação fullstack desenvolvida com backend em Java (Spring Boot) e frontend em React, permitindo cadastro, login e publicação de postagens com autenticação via JWT. O sistema segue os princípios REST e utiliza banco de dados relacional com integração via JPA e MySQL.

O projeto demonstra domínio em toda a stack de desenvolvimento web: desde a construção da API segura e escalável até a criação de uma interface interativa com React, usando rotas protegidas e consumo de APIs. Ideal para apresentar habilidades em desenvolvimento moderno e integração frontend-backend.
